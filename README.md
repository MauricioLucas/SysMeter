# SysMeter by jNizM
Shows ComputerInfo via AutoHotkey

  
## Features
* Shows CPU Usage
* Shows RAM & SWAP Usage
* Shows Drives Usage
* Toggle MB/GB to Percentage
* Toggle Transparency
* Toggle AlwaysOnTop
